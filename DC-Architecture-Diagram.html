GitOps Workflow:

1. Engineer creates/updates config files
2. Opens Pull Request for peer review
3. 2 approvals required (code review)
4. Merge to main branch

5. Engineer manually triggers workflow (workflow_dispatch)
6. Automated deployment begins (~7 min)
